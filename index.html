First post

Posted on March 13 in 2019

Something excited happened. I learned how to do even more font styles than ever before! There is font family, which controls what the font is set at. Font size controls how big you want your font to be. Font style controls extras like italics (only font style i know of right this minute). You can use short hand so you don't have to have multiple lines, but most people forget in which order you need to run the information in. Font weight is great for if you want something to be bold. Line height is great if you want to increase the space between lines. Finally text align sets where you want your text to start in the page.
